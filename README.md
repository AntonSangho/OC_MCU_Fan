## 하드웨어 

- 아두이노 우노  
- 5V 모터 
- 다이오드 : 1N5817
- NPN 트렌지스터: 2N3904
- 가변저항 10k
- M-M 점퍼선 6개 
- USB 2.0 케이블 Type A/B


**주의: 점퍼선 연결시 케이블을 제거하고 진행합니다.**
